## Options

| Key                     | Type                                          |
| ------------------------ | --------------------------------------------- |
| photoSrc               | string, required |
| videoSrc  | string, required |
| loop               |  null | 1 |
| muted               |  null | 1 |
| useApple               |  null | 1 |


## Run

```shell
pnpm install && pnpm run dev
```
